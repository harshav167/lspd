droid-lsp is a Go project that provides an LSP-backed wrapper/daemon and read hook for surfacing diagnostics into Droid sessions. Repo structure centers on cmd/ plus internal packages such as config, daemon, format, log, lsp, mcp, metrics, policy, socket, and watcher. Binaries/build artifacts in repo root include lspd and lsp-read-hook. Config uses YAML for daemon config and JSON for Droid hook/settings integration.
